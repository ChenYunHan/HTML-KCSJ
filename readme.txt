该软件分为两个大的模块：
1.我的
点击底部导航栏的我的可以跳转到我的页面，要使用软件需先登录才能进行操作。
登录操作：点击“登录/注册”按钮，跳转到登录页面，输入正确的用户名和密码完成登录返回我的页面。
查看我的企业：在登陆之后，点击我的企业这一栏，可查看当前用户所拥有的企业。
查看个人信息：在登陆之后，点击我的页面上的电话号码，可以查看当前用户的信息，并且点击信息栏可以对当前信息进行修改。
退出登录：在登陆之后，点击设置，进入设置页面，点击退出登录这一栏，可以退出当前登录的用户。
2.看账
在登陆之后，点击底部导航栏的看账可以跳转到看账页面。
切换企业：点击看账页面上方的下拉框，可以切换企业查看不同信息。
利润情况：点击看账页面的利润情况这一栏，跳转到利润情况页面，可以切换年份查看不同年份的信息，点击不同的标签页可以查看不同情况的折线图。
纳税情况：点击看账页面的纳税情况这一栏，跳转到纳税情况页面，可以切换年份查看不同年份的信息，还可以点击右上角的切换按钮，查看月份和季度的信息。
应收账款：点击看账页面的应收账款这一栏，跳转到应收账款页面，可查看当前年份月份的余额和应收明细。
